Setup - software recommendations for coders

  • Local development environment: XAMPP is an easy to install Apache distribution containing MySQL, PHP and Perl. https://www.apachefriends.org/
  • Text editor: Sublime Text 2: http://www.sublimetext.com/2
  • SSH client: PuTTY http://www.putty.org/
  • FTP/sFTP: Pick one:
    • WinSCP http://winscp.net
    • Filezilla https://filezilla-project.org/
  • Browsers: Must have at least the following
    • Firefox https://www.mozilla.org/ with Firebug https://getfirebug.com/
    • Chrome https://www.google.com/chrome/browser
  • Git: https://windows.github.com/ or http://git-scm.com/downloads
